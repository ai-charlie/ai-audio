# README
## 项目文件说明
- README.md
    - 本项目说明书
    
- VIDEO50 文件夹 
    - 一师一优课2019年50个初中语文教学视频
    - 教学视频字幕文件
    - 教学视频音频文件——通过ffmepg 提取的MP3
    - 对字幕进行的词频统计图
  
- video-srt-gui-ffmpeg-0.3.3-x64 文件夹
    - go语言编写的阿里云的智能语音服务
    - 包括对象存储oss服务和语音文件识别
  
- getWAV.py
    - 将VIDEO50文件夹中的视频提取音频，存入AUDIO50文件夹
  
- word_counts.py
    - 对教学视频的字幕进行词云图绘制
  
- simsun.ttc
    word_counts.py中进行词云绘制的字体文件
  
- ffmpeg.exe
    getWAV.py 中进行音频提取的ffmepg软件
